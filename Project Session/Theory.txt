Steps to deploy Application

Clone this repository
git clone https://github.com/OmkarMemane09/SkyReserve.git

Install Mysql Server and create database
apt update -y
apt install mysql-server -y
mysql_secure_installation
mysql -uroot -p
>> create user linux identified by "Redhat";
>> grant all privileges on \*.\* to linux;
>> flush privileges;
>> create database flightdb;
>> exit


Deploy Backend

cd SkyReserve
cd FlightReservationSystem
apt install openjdk-17-jdk -y
apt install maven -y
export DATASOURCE_URL="jdbc:mysql://localhost:3306/flightdb"
export DATASOURCE_USER="linux"
export DATASOURCE_PASSWORD="Redhat"
export FRONTEND_URL="http://your instance public ip:80"

mvn clean package     # do this where ur pom.xml present

java -jar target/flight*.jar   #do this after ur frontend get deployed bco after this termina get attaches

Deploy Frontend (open new tab)

cd SkyReserve
cd frontend
apt install nodejs npm -y
export VITE_API_URL=http://instance public ip:8080
npm install   # do where ur oackage.json present
npm run build
apt install apache2 -y
cp dist/* /var/www/html/
systemctl start apache2

then do java -jar
